포트폴리오 프로젝트 데이터로 만들어둔 md 파일들을 기반으로,
이 프로젝트를 보여주는 **프로젝트 상세 페이지 UI를 구현해줘.**

---

## 📂 입력 데이터

* 프로젝트 md 파일들: `/content/portfolio_content\` 폴더안에있음
* 이 파일들은 이미 다음과 같은 구조를 가지고 있음:

  * Overview
  * Key Features
  * Technical Highlights
  * Tech Stack
  * Achievements
  * Summary 등

👉 이 md 데이터를 **직접 파싱해서 UI에 렌더링**해야 함

---

## 🧱 기술 스택

* Next.js (App Router)
* TypeScript
* Tailwind CSS

---

## 🎯 목표

* md 데이터를 기반으로 프로젝트 상세 페이지 구현
* 포트폴리오 사이트에서 실제 사용 가능한 수준
* 단순 텍스트 출력 ❌ → 구조화된 UI ⭕

---

## 🧩 구현 요구사항

### 1. 페이지 구조

* `/projects/[slug]/page.tsx` 형태로 구현
* slug 기반으로 md 파일 불러오기

---

### 2. 데이터 처리

* md 파일을 읽어서 파싱하는 로직 포함
* frontmatter + 본문 데이터 모두 활용

---

### 3. UI 구성

본래 UI를 따르되, 더괜찮은 방향이있으면 그방향으로

---

### 4. 디자인 요구

* padding / margin / typography 일관성 유지
* 섹션 간 간격 일정하게 유지
* 너무 화려한 디자인 금지

---

### 5. 중요한 요구

* 단순 map으로 텍스트 찍는 수준 ❌
* 읽기 좋게 hierarchy (제목, 본문, 강조) 표현
* 핵심 내용은 시각적으로 강조

---

### 6. 추가 요구

* 재사용 가능한 컴포넌트 구조로 설계
* 다른 프로젝트 md도 그대로 사용할 수 있게 확장 가능하게 구현

---

## 📦 출력

* 전체 코드 제공
* 파일 구조 포함
* 필요한 유틸 함수 (md 파싱 등) 포함

---

## 🔥 핵심

이 작업은 단순 렌더링이 아니라
👉 “포트폴리오 완성도를 높이는 UI 구현”이다.

md 데이터 구조를 최대한 잘 살려서,
읽는 사람이 “이 개발자 뭐 잘했는지” 빠르게 파악할 수 있게 만들어줘.
